  • Weatherstripping or window seal
  • Utility knife
  • Measuring tape
  • Cleaning solution
  • Clean rags
  • Scraper or putty knife
  • Scissors

Estimated Time: 45-60 minutes

üìù Step-by-Step Instructions

  1. Prepare the window:
    • Clean the window frame thoroughly
    • Remove old weatherstripping
    • Clean off any adhesive residue
  2. Measure for new seal:
    • Measure the length of each side
    • Add 1 inch extra for corners
    • Purchase appropriate weatherstripping
  3. Install new weatherstripping:
    • Cut strips to measured length
    • Remove backing from adhesive
    • Press firmly into place
    • Cut corners at 45-degree angles
  4. Test the seal:
    • Close window completely
    • Check for any gaps
    • Ensure window operates smoothly
  5. Final adjustments:
    • Trim any excess material
    • Press down any loose edges
    • Clean up work area
