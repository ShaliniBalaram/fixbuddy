Replace a Broken Door Knob

Learn how to replace a broken or malfunctioning door knob with a new one.

Tools & Materials

  • New door knob set
  • Screwdriver
  • Measuring tape
  • Chisel (if needed)
  • Wood filler (if needed)
  • Utility knife
  • Pencil
  • Safety glasses

Fix Steps

Step 1: Remove Old Knob

  • Locate mounting screws
  • Remove screws carefully
  • Pull apart knob assembly
  • Remove latch assembly

Step 2: Prepare Door

  • Clean mounting holes
  • Check hole sizes
  • Repair if needed
  • Test fit new hardware

Step 3: Install New Latch

  • Insert latch assembly
  • Align with strike plate
  • Secure faceplate
  • Check operation

Step 4: Mount New Knob

  • Install exterior knob
  • Connect interior piece
  • Secure mounting screws
  • Test functionality

Common Problems

Misaligned Strike Plate

Solution: Adjust plate position or add shims

Wrong Size Knob

Solution: Measure carefully before purchasing replacement

Sticking Latch

Solution: Check alignment and lubricate mechanism

Maintenance Tips

  • Lubricate annually
  • Check screws regularly
  • Clean mechanism
  • Test lock function
  • Keep spare keys
