  • Allen wrench set

Estimated Time: 30-45 minutes

üìù Step-by-Step Instructions

  1. Diagnose the problem:
    • Key won't turn smoothly
    • Lock is sticking
    • Misaligned latch
    • Loose handle
  2. Clean the lock:
    • Use compressed air to remove debris
    • Apply graphite powder into keyway
    • Work key in and out several times
  3. Check alignment:
    • Ensure strike plate aligns with latch
    • Tighten any loose screws
    • Adjust strike plate if needed
  4. Fix loose handle:
    • Remove handle cover plate
    • Tighten mounting screws
    • Check handle spring
  5. Test the repair:
    • Insert and turn key multiple times
    • Check handle operation
    • Verify door latches properly
