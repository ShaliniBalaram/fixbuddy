Fix Door Hinges

Learn how to diagnose and repair common door hinge problems, from squeaky hinges to loose screws.

Safety Warning

Doors are heavy and can cause injury if they fall. Always have someone help you when removing a door from its hinges. Use proper lifting techniques and wear safety gloves. Keep fingers clear of pinch points.

Tools & Materials

  • Screwdriver set
  • Drill with bits
  • Wood dowels
  • Wood glue
  • Replacement hinges
  • WD-40 or hinge lubricant
  • Safety gloves
  • Door shims

Common Problems & Solutions

Squeaky Hinges

  • Clean hinge thoroughly
  • Apply lubricant
  • Work door back and forth
  • Wipe excess lubricant

Loose Screws

  • Remove loose screws
  • Insert wood dowels
  • Apply wood glue
  • Install new screws

Sagging Door

  • Check hinge alignment
  • Tighten all screws
  • Add longer screws
  • Adjust strike plate

Replacing Hinges

  • Support door properly
  • Remove old hinge
  • Clean mounting area
  • Install new hinge

Prevention Tips

  • Lubricate hinges annually
  • Check screws quarterly
  • Clean hinges regularly
  • Avoid door slamming
  • Monitor for rust
  • Keep tools handy
  • Address issues promptly
  • Use door stops

Common Issues

Stripped Screw Holes

Use wood dowels or toothpicks with wood glue to create new grip for screws.

Misaligned Hinges

Use shims to adjust door position and ensure proper alignment.

Rusted Hinges

Clean with steel wool and lubricant, or replace if severely damaged.
