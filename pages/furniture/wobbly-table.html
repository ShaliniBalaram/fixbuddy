Fix a Wobbly Table or Chair

Beginner Friendly 10-15 minutes

Learn how to quickly stabilize wobbly furniture with these simple fixes. Most wobbles can be fixed in minutes with common household items.

Before You Start

Check which leg is uneven
Clean the bottom of all legs
Place on flat surface to test

Common Problems & Solutions

Uneven Leg Length

What's Wrong?

Table or chair rocks back and forth on flat surface

How to Fix It:

  1. Find the short leg by rocking furniture
  2. Clean bottom of all legs
  3. Add felt pads or rubber caps to short leg
  4. Test and add more pads if needed

No felt pads? Use folded cardboard or cork from wine bottle

Loose Leg

What's Wrong?

Leg moves or feels loose at connection point

How to Fix It:

  1. Turn furniture upside down carefully
  2. Look for loose screws or bolts
  3. Tighten all connecting parts
  4. Test stability when done

Screw still loose? Add wood glue to screw holes before re-tightening

What You'll Need

Screwdriver
Or: Coin for large screws
Felt pads
Or: Thick cardboard/cork
Wood glue
Or: Strong tape for temporary fix
