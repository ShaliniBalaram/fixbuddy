Fix a Wobbly Table

Learn how to stabilize and repair wobbly tables to ensure safe and stable use.

Safety Warning

Ensure table is properly supported while working on it. Some tables may be heavy and require assistance to move or flip. Use appropriate safety equipment when cutting or sanding materials.

Tools & Materials

  • Level
  • Furniture shims
  • Wood glue
  • Screwdriver set
  • Measuring tape
  • Sandpaper
  • Wood filler
  • Clean rags

Fix Steps

Step 1: Assess the Problem

  • Place table on level surface
  • Check each leg for stability
  • Identify loose joints
  • Measure any gaps

Step 2: Basic Fixes

  • Tighten all screws
  • Add shims where needed
  • Level table surface
  • Test stability

Step 3: Advanced Repairs

  • Repair loose joints
  • Replace damaged parts
  • Reinforce weak areas
  • Sand rough spots

Step 4: Final Touches

  • Clean all surfaces
  • Apply wood filler if needed
  • Test all repairs
  • Make final adjustments

Maintenance Tips

  • Check stability monthly
  • Keep floor surface clean
  • Rotate table position regularly
  • Avoid dragging table
  • Use furniture pads
  • Address issues promptly
  • Keep hardware tightened
  • Monitor floor levelness
